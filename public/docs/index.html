Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Register New User

This endpoint allows you to register new user for a single tenant.

Supports json, xml, yaml üòé
Example request:
curl --request POST \
    "http://localhost/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"name\": \"ex\",
    \"email\": \"gbode@example.net\",
    \"phone\": \"incidunt\",
    \"password\": \"rtF.Q%gr4a\",
    \"enabled\": false,
    \"password_confirmation\": \"praesentium\"
}"

const url = new URL(
    "http://localhost/api/register"
);

const headers = {
    "Content-Type": "application/json",

List all records

This endpoint allows you to list all permissions of a tenant. It's a really useful endpoint, and you should play around with it for a bit.

Supports json, xml, yaml, text üòé
Example request:
curl --request GET \
    --get "http://localhost/api/v1/permissions?cursor=11&previous=15&limit=5" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "http://localhost/api/v1/permissions"
);

const params = {
    "cursor": "11",
    "previous": "15",
    "limit": "5",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
